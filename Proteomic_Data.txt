The mass spectrometry proteomics data have been deposited to the ProteomeXchange Consortium via the PRIDE [1] partner repository with the dataset identifier PXD018461 and 10.6019/PXD018461.

Raw data files (labeled L7 and l8) were analyzed using MaxQuant version 1.6.3.3 with and using the paramters withing the file "copper_mqpar.xml".
